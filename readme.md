# 操作系统课程设计 - 多级目录FAT文件系统 (终极版)

## 版本特性
本项目为完整的操作系统课程设计实现，包含以下核心特性：

### 1. 核心存储
- **内存映射 (mmap)**: 底层 I/O 核心，模拟物理磁盘操作。
- **FAT 显式链接**: 完整的链式存储管理。
- **内存缓冲**: Buffer Pool + LRU 置换 + Dirty Bit 延迟写回。
- **智能写入**: 支持 Grow/Shrink/Equal 三种写入模式，优化 FAT 链操作。

### 2. 多级目录
- 支持标准路径格式（如 `/home/docs/file.txt`）。
- 根目录与子目录混合存储策略。

### 3. 可视化增强 (UI)
- **磁盘视图**: 包含彩色图例 (Legend)，直观展示 SuperBlock / FAT / Dir / Data / Free 区域。
- **详细列表**: 文件列表新增「起始块号」与「基本权限」字段展示。
